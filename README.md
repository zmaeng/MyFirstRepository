# React 앱
이 프로젝트는 과목별 점수 관리 및 성적 계산을 위한 React 애플리케이션입니다. 사용자는 과목을 추가, 수정, 삭제할 수 있으며, 성적 및 점수를 자동으로 계산하여 통계 정보를 제공합니다.

## 주요 기능
- 과목 추가 및 저장, 삭제 기능 제공
- 과목별 점수 및 성적 자동 계산
- 성적 정보를 표 형태로 정렬하여 표시

## 기술 스택
- React: 사용자 인터페이스 구성
- Bootstrap: 스타일링 및 반응형 디자인
- Git: 버전 관리
